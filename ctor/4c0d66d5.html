ATOM介绍：
Atom 是 Github 专门为程序员推出的一个跨平台文本编辑器，是当前主流的跨平台的三大编辑器(Atom,sublime,vscode)之一。Atom内置对markdown的支持，能够方便的进行解析预览。
Markdown是一种可以使用普通文本编辑器编写的标记语言，通过简单的标记语法，它可以使普通文本内容具有一定的格式。

如何使ATOM能编写.MD文件：

首先我们要安装Atom。去官网下载安装对应你平台的Atom版本，安装完成。

然后打开Atom软件。

打开任意一个.md文件(markdown格式的文件)，也可以自己新建一个.md文件再打开，打开方式为菜单栏File->Open file。

windows下使用快捷键 ctrl + shift + p，打开命令输入框；

输入 markdown preview toggle，enter安装，然后ctrl+shift+m即实现左边编辑，右边实时预览。

我们也可以通过菜单栏Packages->Markdown Preview->Toggle Treview实现左边编辑，右边实时预览。

Atom插件分为内置的和扩展的。其中Markdown Preview是Atom的内置插件。

如果要很好的编写MARKDOWN文件，还要安装下列插件：
windows下使用快捷键 ctrl + shift + p，打开命令输入框，输入 settings view install packages and themes，也可以通过菜单栏File->Setting->Install；

打开搜索框在搜索框依次输入下列插件名，点击Packages按钮，在搜索结果中点击Install按钮即可安装。

MARKDOWN-PREVIEW-PLUS：

该插件在markdown-preview的基础上做了扩展，支持预览实时渲染，支持Latex公式。

ctrl + shift + m 打开markdown实时预览

ctrl + shift + x 开启latex公式支持

同步滚动：
激活 markdown-scroll-sync 插件，可以看到Atom已支持同步滚动功能。同步滚动是Markdown编辑器的必备功能，方便翻阅文档修改时能快速定位到预览的位置。markdown-scroll-sync不仅支持同步滚动，在光标位置发生变更时也会同步滚动，这个功能在很多Markdown编辑器中不具备。

注意：
目前的markdown-scroll-sync不支持markdown-preview-plus，仅支持Atom自带的markdown-preview。

代码增强：

激活 language-markdown 插件。一般的Markdown编辑器提供了代码着色等基本功能，language-markdown除了能给代码着色，还提供了快捷的代码片段生成等功能。

图片粘贴：
在安装插件之前，插入一幅图片需要以下步骤：

截图->为文件命名并保存->整理到当前文件夹下->在.md文件中插入相应命令

我们使用一个图片保存在我们.md本地目录的插件markdown-img-paste插件。图片被保存的.md所在目录的assets文件夹中。

安装完成后插入截图只需要：
使用任意截图工具进行截图（无需保存，复制到剪切板就行），然后ctr+shift+v ，会自动粘贴图片链接，并生成一个空白的title让你填写，并在预览中显示出图片。

表格编辑：

激活 markdown-table-editor 插件。

在行首输入|，然后按Tab，如果|变粉红则创建了表格。|后可以写这个单元格的内容，写完了一个单元格再按|。即创建了一个单元格。

PDF导出：

激活 markdown-themeable-pdf 与 pdf-view。

先保存文件Ctrl+s，然后在编辑区点击 右键->Markdown to PDF或者使用快捷键 ctrl + shift + e可以导出PDF并预览，导出的pdf默认与.md在同一文件夹内。

快捷键整理：

ctrl + shift + p 打开命令输入框

ctrl + shift + m 打开markdown实时预览

ctrl + shift + x 开启latex公式支持

ctrl+s保存文件然后ctrl+shift+e导出pdf并预览。

区块的引用：

语句前加一个>，这句话就是引用，注意>后要一个空格。


引用是一个区块，理论上是应该什么内容都可以放，比如说：标题，列表，引用等等。

引用可以嵌套使用。

分割线：

分割线可以由* - _（星号，减号，底线）这3个符号的至少3个符号表示且不需要连续，中间有空格也可以。建议用减号。

链接：

markdown支持2种链接方式：行内式和参数式，不管是哪一种，链接文字都是用 [方括号] 来标记。

行内式的链接格式是：链接的文字放在[]中，链接地址放在随后的（）中。链接还可以带title属性，在链接地址后面空一格，然后用引号引起来。

参数式后面的标签可以用””，’’,或者()括起来均可以。

bat颜色调配

2019-12-01 /
  • bat配色
 / 点击 / 阅读耗时 1 分钟

第一位底色第二位字色中间无空格　如：color 1F　

0 = 黑色　　　8 = 灰色
1 = 蓝色　　　9 = 淡蓝色
2 = 绿色　　　A = 淡绿色
3 = 浅绿色　　B = 淡浅绿色
4 = 红色　　　C = 淡红色
5 = 紫色　　　D = 淡紫色
6 = 黄色　　　E = 淡黄色
7 = 白色　　　F = 亮白色

Windows用命令打开常用的设置页面和常用快捷键

2020-09-20 /
  • 快捷
 / 点击 / 阅读耗时 5 分钟

Win+R输入以下内容来快捷打开常用设置

compmgmt.msc # 计算机管理

diskmgmt.msc # 磁盘管理

devmgmt.msc # 设备管理

services.msc # 服务管理

taskschd.msc # 计划任务管理

lusrmgr.msc # 本地用户和组(win10用户和组设置已经移到设置和控制面板中)

fsmgmt.msc # 共享文件夹管理

perfmon.msc # 计算机性能监测程序

secpol.msc # 本地安全组策略

gpedit.msc # 本地组策略编辑器

taskmgr # 任务管理

SystemPropertiesAdvanced # 系统属性设置(设置环境变量经常用到, 有点长)

netplwiz # 用户帐户(本机用户)

explorer # windows窗口管理器(没有桌面的时候运行它)

cleanmgr # 磁盘清理工具

certmgr.msc # 证书管理

calc # 计算器

dxdiag # DirectX检查

dcomcnfg # 组件服务

eventvwr # 事件查看器

control # win10经常找不到控制面板就用这个

mstsc # 远程桌面连接

mdsched # windows内存诊断

nodepad # 记事本

mspaint # 画图程序

regedit # 注册表

charmap # 启动字符映射表

OptionalFeatures # windows功能(添加和移除windows功能)

osk # 屏幕键盘

msconfig # 系统配置(主要启动的配置)

msinfo32 # windows系统信息

….
命令行命令:

nslookup # 通过dns解析域名的IP

diskpart # 磁盘管理工具

net

tasklist

taskkill
Win10 快捷键(大部分适用win10以下)
win + Q/S 打开contana和搜索
Win + W 打开windows link
win + E 打开文件资源管理器(计算机)
win + R 运行
win + T 选择光标直接跳到任务栏,在程序中切换
win + U 显示设置
win + I windows设置主页
win + P 多屏显示调节
win + A 通知栏
win + D 临时返回桌面, 再按可恢复
win + F 反馈中心
win + G 游戏录制工具
win + K 右侧连接设备栏
win + L 锁屏
win + X 一些常规设置(右击windows菜单图标)
win + B 光标跳到任务栏的通知区域,可在通知选项中切换
win + M 返回桌面(按shift+win+M可还原, 如果win+M按了两次不可还原)
win + = 打开放大镜(win + =放大, win + -缩小)
win + , 临时看下桌面,且不可操作桌面(win键要一直按住)
win + 1~0 对应顺序打开任务上的程序图标
win + home 将当前窗口以外的程序窗口全部最小化(再按一次恢复)
win + pause 打开计算机属性
win + 上/下 调整当前窗口的最大/最小(上能达到最大化, 下能达到最小化)
win + 左/右 将当前窗口拉到最左/右侧
win + shift + S 局部截图(到剪贴板上)
PrintScreen/SysRq 全局截图(复制到剪贴板保存到oneDrive)(多屏)
win + PrtScn 全局截图(复制到剪贴板并保存到本地%userprofile%\Pictures\Screenshots)
Alt + PrtScn 截取当前窗口(并保存到oneDrive)

win + tab 任务视图(虚拟桌面, 历史任务记录)
ctrl + win + D 新增虚拟桌面,并切换到该桌面
ctrl + win + F4 删除当前的虚拟桌面, 并移到上一个窗口
alt + tab 程序切换
alt + win + D 显示时间日历
ctrl + w 关闭活动窗口

文件[夹]快捷键
Alt + Enter 查看文件[夹]属性(需要选中文件[夹])
F2 重用名
Ctrl + D/Delete 删除文件(优先回收站)
Shift + Delete 直接删除(不经过回收站)

Ctrl + Shift + N 新建文件夹
